# Nexa Game Recommender - Resume Description

Full-stack AI-powered game recommendation platform with real-time streaming data and multi-API game database integration

Includes AI recommendation engine, real-time Twitch viewer tracking, game search autocomplete, and comprehensive game details subsystems

Built with FastAPI and Python 3.9 for high-performance async backend development with ASGI server architecture

Developed React 18 frontend application using modern hooks and component-based architecture for responsive, interactive user interface

Implemented OpenAI GPT-4o integration for intelligent game recommendations based on natural language preferences and contextual understanding

Used aiohttp for asynchronous HTTP client operations to efficiently aggregate data from multiple external APIs concurrently

Integrated RAWG API for comprehensive game database access with 500,000+ titles, including metadata, screenshots, and ratings

Integrated IGDB API for real-time game search and autocomplete functionality with OAuth2 token management

Integrated Twitch API for live streaming data and real-time viewer count tracking to display trending games

Leveraged Pydantic models for type-safe request/response validation and data serialization across API endpoints

Utilized slowapi middleware for rate limiting to protect API endpoints and manage request quotas

Implemented custom rate limiting system with IP-based tracking and configurable request windows for API protection

Used Axios for promise-based HTTP client communication between React frontend and FastAPI backend

Leveraged Tailwind CSS utility-first framework for rapid UI development with custom esports-inspired design system

Utilized Headless UI component primitives to ensure accessible and consistent user interface patterns

Integrated python-dotenv for secure environment variable management and API key configuration

Deployed containerized application using Docker with multi-stage builds and optimized Python 3.9-slim base image

Configured Railway platform for automated CI/CD with health check monitoring and auto-restart policies

Implemented async/await patterns throughout backend for concurrent API calls and improved performance

Built custom API request tracking system with daily/monthly statistics and usage monitoring for RAWG API quota management
